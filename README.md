# Transformer Fault Identification System (AdaBoost)

This application is designed to identify fault points in power transformers using the **AdaBoost** machine learning algorithm based on Dissolved Gas Analysis (DGA) data.

The system processes gas concentrations to determine specific transformer fault points using international standard methods.
* **Algorithm:** AdaBoost (Adaptive Boosting).
* **Diagnostic Methods:** Duval Triangle Method (DTM) and Duval Pentagon Method (DPM).
* **Input Parameters:** $H_2$, $CH_4$, $C_2H_2$, $C_2H_4$, and $C_2H_6$.
